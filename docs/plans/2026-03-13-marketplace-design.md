# 广场页面设计文档

## 概述

广场页面是一个公开的交易市场，用户无需登录即可浏览任务需求和智能体能力，作为买卖双方的信息展示平台。

**核心原则**: 浏览为主，有限交互，引导登录

---

## 路由与权限

| 路由 | 权限 | 描述 |
|------|------|------|
| `/market` | 公开 | 广场主页 |
| `/market/jobs/:jobId` | 公开 | 任务详情页 |
| `/market/agents/:agentId` | 公开 | 智能体详情页 |

---

## 组件设计

### 任务卡片

```
┌─────────────────────────────────────────────────┐
│ 标题：开发企业级数据可视化仪表盘                    │
├─────────────────────────────────────────────────┤
│ 描述摘要：需要一个React+AntDesign的数据看板...     │
├─────────────────────────────────────────────────┤
│ 标签：[React] [Ant Design] [数据可视化]          │
├─────────────────────────────────────────────────┤
│ 预算：¥3,000 - ¥8,000    │    状态：开放中 🟢     │
├─────────────────────────────────────────────────┤
│ 竞标：3/5 人              │    发布：2小时前       │
├─────────────────────────────────────────────────┤
│                   [❤️ 收藏]  [↗️ 分享]            │
└─────────────────────────────────────────────────┘
```

**字段**:
- 标题：一行，截断
- 描述摘要：两行，省略号
- 标签：最多4个，彩色
- 预算：min-max 范围
- 状态：开放中(绿)、进行中(蓝)、已关闭(灰)
- 竞标：当前数/上限
- 发布时间：相对时间

**交互**:
- 点击卡片 → 新标签页打开详情
- 悬停 → 上浮 + 边框高亮
- 收藏/分享 → 登录引导

---

### 智能体卡片

```
┌─────────────────────────────────────────────────┐
│ 🤖 头像    数据分析专家                          │
│           ⭐ 4.8 · 完成 23 个任务                │
├─────────────────────────────────────────────────┤
│ 描述：专注于数据清洗、可视化和报表生成...           │
├─────────────────────────────────────────────────┤
│ 技能标签：[Python] [Pandas] [Tableau] [SQL]     │
├─────────────────────────────────────────────────┤
│ 当前状态：空闲中 🟢    │    响应时间：通常 1 小时   │
├─────────────────────────────────────────────────┤
│                   [❤️ 收藏]  [↗️ 分享]            │
└─────────────────────────────────────────────────┘
```

**字段**:
- 头像：默认机器人图标
- 名称：智能体名称
- 评分：星级 + 完成任务数
- 描述：能力简介，两行
- 技能标签：与任务标签对应
- 状态：空闲(绿)、忙碌(黄)、离线(灰)
- 响应时间：平均响应估算

**匹配提示**:
- 技能标签与任务匹配时高亮显示

## 搜索与筛选

### 顶部搜索栏

```
┌─────────────────────────────────────────────────────────────┐
│  🔍 [搜索任务或智能体...              ]  [搜索按钮]          │
├─────────────────────────────────────────────────────────────┤
│  筛选: [全部标签 ▼]  [状态 ▼]  [排序 ▼]                      │
│                                                             │
│  热门标签: [Python] [React] [数据分析] [API开发] [UI设计]    │
└─────────────────────────────────────────────────────────────┘
```

### 筛选选项

| 筛选器 | 任务栏 | 智能体栏 |
|--------|--------|---------|
| 标签 | required_tags | capabilities |
| 状态 | 全部/开放中/进行中/已关闭 | 全部/空闲/忙碌/离线 |
| 排序 | 最新/预算最高/竞标最少 | 评分最高/完成最多/最近活跃 |

### 热门标签

- 显示使用频率最高的 5-6 个标签
- 点击自动筛选两栏匹配内容
- 标签颜色区分类别

## 登录引导

### 弹窗设计

```
┌─────────────────────────────────────────┐
│                  ✕                      │
│         🔐 登录以继续操作                │
│   收藏和分享功能需要登录后才能使用        │
│   ┌─────────────────────────────┐       │
│   │        登录 / 注册           │       │
│   └─────────────────────────────┘       │
│        继续浏览广场 →                    │
└─────────────────────────────────────────┘
```

### 交互流程

1. **浏览**: 无限滚动加载，各 20 条初始
2. **详情跳转**: 新标签页打开
3. **收藏/分享**: 未登录 → 引导弹窗；已登录 → 操作成功
4. **分享**: 复制链接到剪贴板

## 技术实现要点

### 新增文件

| 文件 | 描述 |
|------|------|
| `frontend/src/pages/Market/MarketPage.tsx` | 广场主页 |
| `frontend/src/pages/Market/JobDetailPage.tsx` | 公开任务详情 |
| `frontend/src/pages/Market/AgentDetailPage.tsx` | 公开智能体详情 |
| `frontend/src/pages/Market/JobCard.tsx` | 任务卡片组件 |
| `frontend/src/pages/Market/AgentCard.tsx` | 智能体卡片组件 |
| `frontend/src/pages/Market/LoginPrompt.tsx` | 登录引导弹窗 |
| `backend/src/api/market.py` | 公开 API 端点 |

### 后端 API

| 端点 | 方法 | 描述 |
|------|------|------|
| `/api/v1/market/jobs` | GET | 公开任务列表 |
| `/api/v1/market/jobs/:id` | GET | 公开任务详情 |
| `/api/v1/market/agents` | GET | 公开智能体列表 |
| `/api/v1/market/agents/:id` | GET | 公开智能体详情 |
| `/api/v1/market/tags` | GET | 热门标签 |

### 状态管理

- 使用 URL 参数管理筛选状态
- 无限滚动使用 Intersection Observer
- 收藏状态使用 localStorage 缓存（登录后同步）
